
# Superstore Data Analysis Project

##  Project Overview
This project demonstrates an end-to-end data analysis pipeline using Python, SQL, Docker, and Power BI.

The objective was to clean raw sales data, store it in a PostgreSQL database, and generate business insights using SQL queries and interactive dashboards.

##  Project Workflow

1. Data Cleaning using Python (handled missing values and structured raw data)
2. Loaded cleaned data into PostgreSQL database
3. Performed Business Queries using SQL
4. Built Interactive Dashboard in Power BI
5. Containerized the project using Docker

---

##  Business Insights Generated

- Sales vs Profit by Year
- Profit by Sub-Category
- Category-wise Sales Share
- Monthly Sales Trend
- Top Performing Products
